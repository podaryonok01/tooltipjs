# UI-TOOLTIPJS
##### A simple library that creates custom tooltips on elements.
## Usage
### Add JS

####
```javascript
    import Tooltip from 'ui-tooltipjs';
    const myTooltip = new Tooltip({
        rootElement: document.body,
        tooltipClass: 'tooltip-custom',
        margin: 10,
        position: 'top-center',
    });
```
